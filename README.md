#### 🔗 GOOGLE AGENT DEVELOPMENT KIT **ADK** FRAMEWORK

This folder contains several projects that showcase the power of using the google Agent Development Kit (ADK) and other popular frameworks to build and develop powerful agentic systems

The first step is to create a virtual environment and install the dependencies found in the **requirements.txt** file 

> python -m venv venv \
> pip install -r requirements.txt

Setting up the API key to be used in the project 

In the case of google projects, 
 - create an account with google cloud
 - Create a new project within google cloud
 - Go to google AI studio, create an API key and assign it to your new project 
 - Connect your project to a billing account 
 - Copy your API key and add it to your environment variables
